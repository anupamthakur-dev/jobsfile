# JobsFile

JobsFile is a lightweight job application tracker built for job seekers who are tired of losing context after applying to multiple roles. It stores job descriptions once, tracks every follow-up, and helps you write better follow-up and cold emails without repetitive copy-pasting.

Track jobs. Remember context. Follow up better.

## 🚀 Why JobsFile?

When you apply to many jobs:

- Job descriptions get lost
- Follow-ups become awkward or delayed
- Writing emails means copy-pasting the same details again and again
- You forget where you applied and what you already did

JobsFile solves this by treating the job description as first-class data and tying it directly to follow-ups, reminders, and application history.

## 🧠 Core Features
### 📋 Job Application Tracking

- Save company, role, job description, link, recruiter email
- Track status: Applied, Followed Up, Interview, Rejected, Offer
- Star important applications

### 🧠 Context Preservation

- Store the full job description once
- View it anytime while following up or preparing for interviews
- ✉️ Smart Follow-Ups
- Generate follow-up and cold emails using the saved job description
- Multiple tones: polite, short, cold
- Copy-to-clipboard for quick sending

### ⏰ Follow-Up Reminders (yet not built because of minimum timeline)

- See which applications need follow-up today
- Timeline view of all actions taken on a job

### 📊 Clear Dashboard

- Overview of total applications, pending follow-ups, interviews, offers
- Focus on “what should I do next?”

## 🎯 Built For

- Job-seeking developers (especially juniors)
- Students applying for internships or placements
- Anyone managing multiple applications at once

```
This is not a job portal or a CRM.
It’s a memory + follow-up assistant.
```

## 🛠 Tech Stack

- Frontend: React + Vite
- Styling: Tailwind CSS v4 (OKLCH-based theme)
- UI Components: shadcn/ui
- Icons: Lucide
- State / Storage: Local storage (MVP)
  (Can be extended to Supabase or any backend)

## 🎨 Design System

- Emerald accent theme (OKLCH)
- Light & dark mode support
- Minimal, calm, productivity-focused UI
- Accessibility-conscious contrast choices

## 🧪 Project Scope

JobsFile is intentionally small and focused:
- No email integrations
- No inbox scraping
- No heavy automation

The goal is clarity, not complexity.
